## Examples

```c
#if !defined GetGravity
    native Float:GetGravity();
#endif

```

## Notes

::: warning

Cette fonction n'est pas définie par défaut. Ajouté 'native GetGravity();' juste en dessous de l'include a_samp.inc pour pouvoir l'utiliser.
