# floatround

<TagLinks />

## Description

Round a floating point number to an integer value.

| Name   | Description                   |
| ------ | ----------------------------- |
| value  | The value to round.           |
| method | The floatround method to use. |

## Returns

The rounded value as an integer.

## Examples

```c
floatround(3.3, floatround_ceil);
```
